MasterMind
This is a number-based puzzle game inspired by the classic MasterMind board game. Players must use logic and deduction to discover a hidden 4-digit code.
Game Rules:

A secret 4-digit number is randomly generated at the start
Players have a maximum of 12 attempts to guess the code
Successfully guess the number within 12 tries to win
If all attempts are exhausted without finding the code, the game is lost
